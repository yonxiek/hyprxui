# dots4hypr | created by yonxiek

Мой минималистичный чёрно-белый сетап для **Hyprland**. Основная идея — ничего лишнего, только продуктивность и чистый вид.

---

## Мой сетап:

| Компонент | Что используется |
| :--- | :--- |
| **ОС** | Arch Linux with CachyOS kernel |
| **ОМ** | [hyprland](https://hyprland.org/) |
| **Бар** | [waybar](https://github.com/Alexays/Waybar) |
| **Оболочка консоли** | Zsh + Oh My Zsh |
| **Терминал** | Kitty |
| **Редактор кода** | NeoVim (Custom friend's config) |

## Особенности и фишки:

### 🔹 Hyprland:
- **Кастомные бинды**: Полностью пересмотренная раскладка для быстрого управления окнами и воркспейсами.
- **Внешний вид**: Настроенные отступы (gaps), скругления и тонкие границы окон в монохромном стиле.
- **Layout**: Умное управление окнами (Dwindle layout) для максимального удобства.

### 🔹 Terminal (Kitty):
- **Blur & Transparency**: Настроенное размытие заднего фона для глубокого погружения.
- **Цветовая схема**: Строгий чёрно-белый конфиг, который не отвлекает от работы.
- **Производительность**: Минимальные задержки ввода.

### 🔹 NeoVim:
- **Optimized for Coding**: Кастомный конфиг (special thanks to my friend - [retrojan](https://github.com/retrojan)).
- **Minimalism**: Интерфейс очищен от лишнего мусора, фокус только на коде.

### 🔹 Shell :
- **Black & White Theme**: Кастомная тема с красивыми разделителями.
- **Git Integration**: Визуальное отображение веток и статуса прямо в строке ввода.
- **Status Indicators**: Цветовая индикация успеха выполнения команд.

### 🔹 Waybar:
- **Компактность**: Воркспейсы и системные ресурсы занимают минимум места.
- **Информативность**: Весь важный мониторинг (CPU, MEM, Temp) перед глазами.
- **Control**: Удобное раздельное управление аудио-входами и выходами.

## Установка:

**1. Скопируй данный репозиторий на свой диск:**
```bash
git clone https://github.com/yonxiek/dotsidk.git
cd dotsidk
```
**2. Запусти скрипт для автоматической настройки:**
```bash
sh install.sh
```
